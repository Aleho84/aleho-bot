## Features
🤖 Bot commands

    /start : Activate bot.
    /stop : Deactivate bot.
    /damejuegos : Free games!
    /alehoserver : Show aleho-server status.
    /showlogs  : Show Bot logs.
    /dolarhoy  : Show Dolar price in Argentina market.

Every day it will search and notify you if there is a new game.
